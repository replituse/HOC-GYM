# Instagram Videos

## How to Download and Add Videos

To display the Instagram videos on your website without the profile information:

### Step 1: Download the Videos

Use one of these free online tools to download the Instagram reels:
- **SnapInsta**: https://snapinsta.app
- **SaveInsta**: https://saveinsta.app
- **InstagramSave**: https://instagramsave.net

### Step 2: Download Each Video

### Step 3: Upload Videos

Upload the downloaded videos to this folder (`attached_assets/instagram_videos/`).

### Step 4: Update Code

Once videos are uploaded, I'll update the code to display them using HTML5 video players instead of placeholders.
